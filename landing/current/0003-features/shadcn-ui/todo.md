# RetreatsOS Features Page - Development Plan

## Design Guidelines

### Design References (Primary Inspiration)
- **Linear.app**: Clean, minimal, premium tech aesthetic
- **Notion.com**: Organized content blocks, clear hierarchy
- **Stripe.com**: Professional, trustworthy, elegant spacing
- **Style**: Premium Minimalism + Dark Mode + Tech Luxury

### Color Palette
- Primary: #000000 (Pure Black - background)
- Secondary: #1A1A1A (Dark Gray - cards/sections)
- Accent: #FFFFFF (White - text and highlights)
- Subtle: #404040 (Medium Gray - secondary text)
- Border: #2A2A2A (Subtle borders)

### Typography
- Heading1: Inter font-weight 700 (48px)
- Heading2: Inter font-weight 600 (36px)
- Heading3: Inter font-weight 600 (24px)
- Body/Normal: Inter font-weight 400 (16px)
- Body/Emphasis: Inter font-weight 500 (16px)
- Navigation: Inter font-weight 500 (14px)

### Key Component Styles
- **Buttons**: White background, black text, rounded-lg, hover: slight gray
- **Cards**: Dark background (#1A1A1A), subtle border, rounded-xl
- **Bento Grid**: Large visual blocks with generous spacing
- **Icons**: Thin, sophisticated, minimal style

### Layout & Spacing
- Hero section: Full viewport height with centered content
- Feature blocks: Bento-grid layout with alternating layouts
- Section padding: 120px vertical, generous whitespace
- Card hover: Subtle lift with smooth transitions
